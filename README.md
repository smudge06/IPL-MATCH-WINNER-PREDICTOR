# IPL-MATCH-WINNER-PREDICTOR
# 🏏 IPL Match Winner Predictor

## 📖 Description

This project is a Machine Learning-based web application that predicts the winner of an IPL match using historical match data.

The model is trained using a Random Forest Classifier and considers key match features such as teams, toss winner, and venue to make predictions.

---

## ⚙️ Features

* Predict IPL match winners in real-time
* Clean and interactive UI built with Streamlit
* Team-based inputs with dropdown selection
* Displays predicted winner with team slogan
* Visual enhancements including background and animations
* Approximate model accuracy: **78%**

---

## 🧠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

---

## 📊 Model Details

* Algorithm: **Random Forest Classifier**
* Features used:

  * Team 1
  * Team 2
  * Toss Winner
  * Venue
* Accuracy: **~78%**

## 🚀 How to Run Locally

1. Clone the repository:

   ```
   git clone https://github.com/your-username/ipl-ai-predictor.git
   ```

2. Navigate to the project folder:

   ```
   cd ipl-ai-predictor
   ```

3. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

4. Run the application:

   ```
   streamlit run app.py
   ```
